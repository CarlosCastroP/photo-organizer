# photo-organizer

Organizador de fotos escrito en Python basado en datos EXIF. Puede organizar en base a fecha y a localización

## Dependencias / Dependecies
* Python 3
* Pillow
* geopy

```
pip install Pillow
```

```
pip install geopy
```

## Ejecución del Programa / Executing Program

```
./organizer-gui.py
```
o / or
```
python organizer-gui.py
```
